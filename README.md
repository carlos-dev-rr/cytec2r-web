# Cytec2r Website

This project is an informative website about the company **Cytec2r**, built using **Astro**.  
It is designed to present the company's services in a fast, accessible, and optimized format.

---

## Main Features
- Fast and optimized thanks to Astro
- Fully responsive design
- Reusable and modular components
- Simple to modify and extend

##  Installation & Usage

### **Run the project locally**
If someone wants to run the website locally in a development environment:
```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
npm install
npm run dev     # Start development server (usually at: http://localhost:4321) 
npm run build   # Build the project for production
npm run preview
```

## Project Structure
```bash
/
├── public/              # Static assets
├── src/
│   ├── components/      # Reusable Astro/JS components
│   ├── layouts/         # Page layouts
│   ├── pages/           # Website pages (routes)
|   ├── scripts/         # Logic functions (JS)
│   └── styles/          # Global or component-specific styles
├── package.json         # Project metadata & scripts
├── astro.config.mjs     # Astro configuration
└── tailwind.config.js   # TailwindCSS configuration
```
